Инструменты для работы с данными

Pattern Matching
a.k.a. switch выражение на стероидах
Destructuring
извлечение данных из ADT со специальным синтаксисом

Pattern Matching

def fizzbuzz(n) do
  case {rem(n, 3), rem(n,5)} do
    {0, 0} -> "FizzBuzz"
    {0, _} -> "Fizz"
    {_, 0} -> "Buzz"
    _ -> n
  end
end

Образ мышления фп

  • Начинайте с модели данных
  • Напишите действия
  • Выделите логику в чистые функции, протестируйте
  • Повторите

Функциональное вносит простоту

  • Вычисления вместо действий
  • Значения вместо переменных
